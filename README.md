[documentation](https://lazyvim.github.io/installation)

## 📥 Installation

```bash
git clone --depth 1 -- https://github.com/rexackermann/LazyVim ~/.config/nvim
```

```bash
# avoid this one
git clone --depth 1 -- git@github.com:rexackermann/LazyVim.git ~/.config/nvim
```
